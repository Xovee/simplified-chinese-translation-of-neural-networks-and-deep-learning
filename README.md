《神经网络与机器学习》是一本免费的在线书籍。这本书的内容包括：
- 神经网络，一种受生物学启发的编程方式，可以让计算机从观察到的数据中进行学习
- 深度学习，一种在神经网络中非常强大的技术

神经网络和深度学习最近在许多领域都取得了非常不错的成绩，例如在图像识别、语音识别，以及自然语言处理等。这本书将会教给你许多神经网络和深度学习之中的核心概念。
